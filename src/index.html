¿Quiénes somos?

Somos dos estudiantes de Laboratoria que están interesadas en aprender a programar en Java Script para desarrollar páginas web. Nos interesan los proyectos sociales, ya que estamos convencidas de que la tecnología puede contribuir a generar impactos positivos en la sociedad.
Este proyecto tiene el propósito de que estudiantes de las ciencias sociales y humanidades accedan con rapidez y seguridad a la base de datos del Banco Mundial para facilitar las investigaciones relacionadas con las mujeres en América Latina.

¿Qué hacemos?

Desarrollamos páginas web para formar nuevas maneras de interactuar con la sociedad.
